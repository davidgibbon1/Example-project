# Portfolio Website

A fast, modern portfolio website built with Next.js and MDX, featuring a clean design system, dark mode support, and excellent performance scores.

## Overview

This portfolio website serves as a showcase for my projects, writing, and professional experience. It's built with modern web technologies and follows best practices for performance, SEO, and user experience.

## Design Philosophy

The design focuses on:
- **Typography-first approach**: Clean, readable content
- **Minimal aesthetics**: Let the work speak for itself  
- **Accessibility**: WCAG 2.1 AA compliance
- **Performance**: Fast loading and smooth interactions

## Technical Features

### Content Management
- **MDX Integration**: Write content in Markdown with React components
- **Type Safety**: Full TypeScript support for content schemas
- **Hot Reloading**: Instant preview during development
- **Git-based**: Version controlled content workflow

### Performance Optimizations
- **Static Generation**: Pre-built pages for fast loading
- **Image Optimization**: Next.js automatic image optimization
- **Font Loading**: Optimized web font loading strategy
- **Code Splitting**: Automatic route-based code splitting

### User Experience
- **Dark Mode**: System preference with manual toggle
- **Responsive Design**: Mobile-first approach
- **Smooth Animations**: Subtle motion with Framer Motion
- **Fast Navigation**: Optimized routing and prefetching

## Architecture

### File Structure
```
portfolio/
├── app/                 # Next.js app directory
├── components/          # Reusable UI components
├── content/            # MDX content files
├── lib/                # Utility functions
└── public/             # Static assets
```

### Content Schema
Each project includes structured frontmatter:
- Title, description, and status
- Technology stack and tags
- Links to demo, repository, case study
- Publication date and featured status

### Component System
Built with shadcn/ui components:
- Consistent design tokens
- Accessible by default
- Customizable with CSS variables
- Dark mode support built-in

## Development Workflow

### Content Creation
1. Write MDX files in `/content` directory
2. Include frontmatter with metadata
3. Use custom components for rich content
4. Preview changes with hot reloading

### Deployment
- **Platform**: Vercel for seamless Next.js deployment
- **CI/CD**: Automatic builds on git push
- **Domain**: Custom domain with SSL
- **Analytics**: Privacy-friendly analytics integration
